這包是「班表模板＋個人固定班表＋特例日期班表」版本。

【要放到 GitHub 的檔案】
- index.html
- dashboard.html
- clock.html
- leave.html
- parttime.html
- register.html
- task.html
- schedule.html  ← 新增頁面
- style.css
- app.js
- config.js

【要放到 Apps Script 的檔案】
- apps-script.gs

【部署前要做】
1. 在 Apps Script 貼上新的 apps-script.gs
2. 儲存後先執行一次 setupSheets()
3. setupSheets() 會自動建立以下工作表，並在 A1 放好標題列，不需要你手動建：
   - 班表模板
   - 個人班表設定
   - 個人固定班表
   - 特例日期班表
4. setupSheets() 也會自動補入預設模板：
   - 專任：星期二到星期五 12:30-21:00、星期六星期日 10:00-21:00
   - 工讀：星期二與星期四 18:00-21:00
5. 重新部署 Web App
6. 再把 GitHub 檔案覆蓋上去

【如果你想手動建立工作表，A1 要貼什麼】
一、班表模板
A1: 規則ID	員工類型	星期	上班時間	下班時間	是否啟用	是否計算遲到	備註	建立時間	更新時間

二、個人班表設定
A1: 設定ID	員工ID	員工姓名	員工Email	是否啟用個人班表	備註	建立時間	更新時間

三、個人固定班表
A1: 規則ID	員工ID	員工姓名	員工Email	星期	上班時間	下班時間	是否啟用	是否計算遲到	備註	建立時間	更新時間

四、特例日期班表
A1: 規則ID	員工ID	員工姓名	員工Email	日期	上班時間	下班時間	是否啟用	是否計算遲到	備註	建立時間	更新時間

【判斷順序】
1. 特例日期班表
2. 個人固定班表（需先在個人班表設定把「是否啟用個人班表」開成「是」）
3. 班表模板（專任或工讀）
