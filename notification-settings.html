LINE 主管通知收件人設定

這裡只設定「主管通知要送到哪一個 LINE」。它不會新增員工、不會影響薪資、班表、打卡、請假或外聘老師資料。
柚子樂器官方 LINE 是發訊息出去的帳號；下方填的是接收主管通知的 LINE User ID。員工或老師回覆 LINE 時，仍會回到柚子樂器官方 LINE 聊天室。

主要主管 LINE 收件人

目前只保留一個主要收件人。之後如果要改成其他人收，只要把這裡的 LINE User ID 換掉並重新儲存即可。

通知項目設定

這一區是「什麼事情要通知」。主管收件人固定看上方主要 LINE 設定；各模組的通知項目仍分開管理。
簽核 / 打卡 / 請假提醒註冊、補打卡、打卡修正、遲到提醒、請假、工讀時數與臨時出勤。 外聘老師提醒外聘合約、合約未簽、拿貨詢價與主管回覆詢價。 證明提醒在職證明、教學證明送出、核准、退回與逾期未審。 應聘老師提醒應聘資料送出、初審、退回、補件與逾期未審。
