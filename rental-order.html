租用規則說明

【租用流程】 填寫租用表格 → 加入柚子樂器官方 LINE → 等待官方回覆 → 確認配送或自取時間 → 簽署合約 → 付款後安排配送 / 安裝。 【退租流程】 透過官方 LINE 或合約查詢頁申請退租 → 確認收回日期 → 安排收回 → 完成退租。 【重要說明】 電鋼琴與電子鼓一期為 90 天。實際型號、租金、運費與配送時間，仍以主管確認後的正式合約為準。
