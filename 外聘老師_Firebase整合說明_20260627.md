# 外聘老師 Firebase 新流程整合說明 2026-06-27

這版已經直接整合到目前專案結構，不使用 `.gs`。

## 已新增檔案

- `external-teacher-onboarding.html`
  - 外聘老師資料填寫、LINE 綁定、身分證明上傳、契約確認、線上簽名。

- `external-teacher-payroll.html`
  - 外聘老師補填薪資／匯款資料。

- `external-teacher-contract-admin.html`
  - 管理端修改外聘才藝教師委任契約條文，每一條可單獨修改。

  - Firebase Functions 新後端。

- `storage.rules`
  - 補上 GitHub Actions 部署 `storage` 時需要的 Storage rules。

## 新資料位置

外聘老師本人資料：

```text
employees/{teacherId}.externalTeacher
```

LINE 綁定碼：

```text
externalTeacherLineBindings/{bindCode}
```

契約模板與條文：

```text
externalTeacherContractTemplates/{templateId}
externalTeacherContractTemplates/{templateId}/clauses/{clauseId}
```

已簽合約：

```text
externalTeacherContracts/{contractId}
```

檔案索引：

```text
externalTeacherFiles/{fileId}
```

Storage 檔案：

```text
externalTeachers/{teacherId}/identity/...
externalTeachers/{teacherId}/signatures/...
externalTeachers/{teacherId}/contracts/...
```

## LINE 綁定指令

老師到官方 LINE 貼上：

```text
外聘老師綁定 EXT-ABC123
```

成功後會回覆老師：

```text
外聘老師 LINE 綁定完成 ✅

您好 {{老師姓名}}，系統已完成您的 LINE 綁定。

請回到剛剛的外聘老師資料填寫頁，繼續完成基本資料、身分證明文件與契約簽署。
```

## 銀行資料略過

老師簽約時可以略過銀行資料。

略過後：

```text
employees/{teacherId}.externalTeacher.payrollInfoStatus = pending
```

排程每天 10:00 檢查一次；若距離上次提醒滿 3 天，會 LINE 提醒補填。

## 部署

照原本 GitHub Actions push main 即可，或手動：

```bash
firebase deploy --only hosting,firestore:rules,storage,functions --project youzi-c1b74 --force
```
